### Q7. 오늘 하루를 예로 들어 업무 흐름을 처음부터 끝까지 적어주세요. ★

**답변 상태**: 부분 확보

**출처: CS 현황 + Atlas 사용 시도 기록**

현재 CS 업무 흐름:
```
1. oqupie에 접속하여 신규 티켓 확인
2. 각 티켓별 페이지에 진입하여 추가필드(영수증 번호, 유저번호 등) 확인
   - 슬랙/텔레그램 연동으로는 추가필드 확인 불가, 반드시 oqupie 직접 접속 필요
3. backnd console(뒤끝)에 접속하여 유저 데이터 조회/조작
   - 다중 하위 페이지 및 필드 진입 필요
   - csv 차트 조회 빈번
4. 필요 시 Google Cloud 로그 탐색기에서 헌터/용사 로그 확인
5. 필요 시 BigQuery에서 기타 프로젝트 로그 확인
6. 조치 수행 후 고객 응답
```

**미비 사항**: 아침 첫 행동, 집중 시간대, 산만한 시간대, 하루 처리 건수 등 미확인
